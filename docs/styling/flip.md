# Flip

Flip your icons vertically or horizontally using the `is-flip-{direction}` classes.

**Class format:** `is-flip-{direction}`

| Class | CSS |
| --- | --- |
| `is-flip-v` | `transform: scaleY(-1);` |
| `is-flip-h` | `transform: scaleX(-1);` |

**Example:**
```html
<i class="if-heart is-flip-v"></i>
```

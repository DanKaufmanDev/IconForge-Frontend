### Arbitrary Values

> **Note:** Arbitrary values are supported for all gap utilities.

You can also use arbitrary gap values using the following syntax:

**Class format:** `is-gap-[{value}]`

**Example:**
```html
<div class="is-grid is-gap-[1rem]">
  <!-- ... -->
</div>
```

| Class | CSS |
| --- | --- |
| `is-text-start` | `text-align: start;` |
| `is-text-end` | `text-align: end;` |
| `is-text-left` | `text-align: left;` |
| `is-text-right` | `text-align: right;` |
| `is-text-center` | `text-align: center;` |
